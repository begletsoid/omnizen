# Project Brief

## Цель
Персональный веб-дашборд в тёмной теме, где пользователь управляет привычками, проблемами/решениями, микрозадачами и визуальными виджетами. Макет должен быть drag-and-drop, состояние сохраняется в Supabase и доступно после авторизации через Google.

## Обязательные виджеты
- Лента привычек со статусами (зелёный/жёлтый/красный), поддержкой drag-and-drop и динамической высотой.
- Таблица «Проблемы / Решения» с управлением строками и вариантами.
- Список микрозадач с отметкой выполнения и сортировкой drag-and-drop.
- Виджет-картинка с загрузкой изображений и изменением размера.

## Технические опоры
- Frontend: React + Vite + TypeScript, тёмная тема по умолчанию, управление состоянием (например, Zustand).
- Хранение и auth: Supabase (Postgres + Storage) с входом через Google.
- Хостинг: Netlify (auto deploy из GitHub).

## Критерии успеха
1. Авторизация через Google, все данные персональны.
2. Дашборд и виджеты настраиваются drag-and-drop, раскладка сохраняется.
3. CRUD-операции для всех виджетов, изменения мгновенно отображаются.
4. Деплой на Netlify с CI из Git.
