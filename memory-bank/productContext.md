# Product Context

- **Проблема**: пользователю нужен единый экран для слежения за привычками, проблемами, решениями и задачами без переключения между приложениями.
- **Пользователь**: один человек (позже — несколько аккаунтов) работает ежедневно, ценит быстроту правок и визуальное представление прогресса.
- **Опыт**:
  - Всегда тёмная тема, минимум отвлекающих элементов.
  - Drag-and-drop управление виджетами и их внутренними элементами.
  - Инлайн-редактирование текста, минимум модалок.
- **Хранение**:
  - Все данные сохраняются в Supabase (таблицы + storage).
  - Привязка к профилю через Google auth, позволяя заходить с разных устройств.
- **Фичи ближайшего релиза**:
  1. Минимальный набор виджетов (привычки, проблемы/решения, задачи, картинка).
  2. Управление расположением виджетов и базовые CRUD-операции.
  3. Синхронизация между сессиями (layout + содержимое).
